Documentación básica para desarrolladores web.

Teoria basica: HTML

Título: "Introducción al Desarrollo Web con HTML"

Recopilación digital introductória al lenguaje de marcado HTML (HyperText Markup Language). A medida que avances, explorarás los conceptos fundamentales de HTML y aprenderás cómo estructurar y presentar contenido en la web.

Capítulo 1: Fundamentos de HTML

En este capítulo, te familiarizarás con los fundamentos de HTML. Aprenderás sobre la estructura básica de un documento HTML y cómo utilizar etiquetas, elementos y atributos para marcar el contenido.

<!DOCTYPE html>
<html>
<head>
  <title>Mi primera página HTML</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Esta es mi primera página web.</p>
</body>
</html>

Capítulo 2: Elementos de texto en HTML

En este capítulo, exploraremos los elementos de texto en HTML. Aprenderemos cómo crear encabezados, párrafos, enlaces y listas para estructurar y dar formato al contenido de nuestra página web.

<h1>Título principal</h1>
<p>Este es un párrafo de ejemplo.</p>
<a href="https://www.ejemplo.com">Enlace a Ejemplo.com</a>
<ul>
  <li>Elemento de lista 1</li>
  <li>Elemento de lista 2</li>
  <li>Elemento de lista 3</li>
</ul>

Capítulo 5: Formularios en HTML

Teoria basica: CSS

Título: "Fundamentos de CSS para Diseño Web"

Recopilación digital introductória al lenguaje de CSS, explorarás los fundamentos de CSS (Cascading Style Sheets) y aprenderás a aplicar estilos a tus páginas HTML para mejorar su apariencia y diseño. A medida que avances, descubrirás cómo utilizar selectores, propiedades y valores de CSS para personalizar la apariencia de los elementos HTML.

Capítulo 1: Introducción a CSS

En este capítulo, aprenderás los conceptos básicos de CSS y cómo enlazar hojas de estilo externas a tus documentos HTML. También descubrirás cómo aplicar estilos a elementos HTML utilizando selectores de CSS.

<link rel="stylesheet" href="styles.css">

Capítulo 2: Selectores de CSS

En este capítulo, aprenderás diferentes tipos de selectores de CSS, como selectores de elemento, selectores de clase y selectores de ID. Estos selectores te permiten apuntar a elementos específicos en tu documento HTML y aplicar estilos personalizados.

<style>
/* Selector de elemento */
h1 {
  color: blue;
}

/* Selector de clase */
.red-text {
  color: red;
}

/* Selector de ID */
#my-button {
  background-color: green;
}
</style>

Capítulo 3: Modelos de caja y posicionamiento

En este capítulo, descubrirás cómo funciona el modelo de caja de CSS y cómo puedes controlar el tamaño, el margen y el padding de los elementos. También aprenderás diferentes técnicas de posicionamiento, como posicionamiento relativo, absoluto y fijo.

<style>
/* Modelo de caja */
.box {
  width: 200px;
  height: 200px;
  border: 1px solid black;
  margin: 10px;
  padding: 20px;
}

/* Posicionamiento relativo */
.relative-box {
  position: relative;
  top: 20px;
  left: 30px;
}

/* Posicionamiento absoluto */
.absolute-box {
  position: absolute;
  top: 50px;
  left: 100px;
}

/* Posicionamiento fijo */
.fixed-box {
  position: fixed;
  top: 10px;
  right: 10px;
}
</style>

Capítulo 4: Diseño de diseño flexible y receptivo

En este capítulo, aprenderás sobre técnicas de diseño flexible y receptivo en CSS. Descubrirás cómo utilizar el diseño basado en rejilla y los medios de consulta (media queries) para crear diseños que se adapten a diferentes tamaños de pantalla.

<style>
/* Diseño basado en rejilla */
.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

/* Medios de consulta */
@media screen and (max-width: 768px) {
    .container {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media screen and (max-width: 480px) {
    .container {
        grid-template-columns: 1fr;
    }
}
</style>

Capítulo 5: Tipografía y estilos de texto

En este capítulo, explorarás cómo dar estilo a la tipografía y al texto en CSS. Aprenderás a cambiar el tamaño de la fuente, el tipo de fuente, el color y otros estilos relacionados con el texto.

<style>
/* Cambiar tamaño de fuente */
h1 {
    font-size: 24px;
}

/* Cambiar tipo de fuente */
p {
    font-family: Arial, sans-serif;
}

/* Cambiar color de texto */
.span-text {
    color: #FF0000;
}

/* Estilos adicionales */
.italic-text {
    font-style: italic;
}

.underline-text {
    text-decoration: underline;
}
</style>

Capítulo 6: Efectos y animaciones CSS

En este capítulo, descubrirás cómo aplicar efectos y animaciones a elementos utilizando CSS. Aprenderás a agregar transiciones, transformaciones y animaciones para mejorar la experiencia visual de tu sitio web.

<style>
/* Transiciones */
.button {
    transition: background-color 0.3s ease;
}

.button:hover {
    background-color: #00FF00;
}

/* Transformaciones */
.rotate {
    transform: rotate(45deg);
}

.scale {
    transform: scale(1.2);
}

/* Animaciones */
@keyframes move {
    0% {
        transform: translateX(0);
    }
    100% {
        transform: translateX(100px);
    }
}

.box {
    animation: move 2s infinite alternate;
}
</style>

Capítulo 7: Fondos y gradientes

En este capítulo, aprenderás a aplicar fondos y gradientes a elementos utilizando CSS. Descubrirás cómo cambiar el color de fondo, agregar imágenes de fondo y crear gradientes de colores suaves o de múltiples colores.

<style>
/* Cambiar color de fondo */
.container {
    background-color: #F0F0F0;
}

/* Agregar imagen de fondo */
.body-bg {
    background-image: url('background.jpg');
    background-size: cover;
}

/* Gradiente lineal */
.linear-gradient {
    background-image: linear-gradient(to right, #FF0000, #0000FF);
}

/* Gradiente radial */
.radial-gradient {
    background-image: radial-gradient(circle, #FFFF00, #FF00FF);
}
</style>

Capítulo 8: Flexbox y diseño de cuadrícula

En este capítulo, explorarás las características de Flexbox y las capacidades de diseño de cuadrícula en CSS. Aprenderás a crear diseños flexibles y responsivos utilizando estas técnicas de diseño modernas.

<style>
/* Flexbox */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

.item {
    flex: 1;
    margin: 10px;
}

/* Diseño de cuadrícula */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-gap: 10px;
}

.grid-item {
    background-color: #F0F0F0;
    padding: 10px;
}

/* Consultas de medios para diseño responsivo */
@media screen and (max-width: 768px) {
    .grid-container {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media screen and (max-width: 480px) {
    .grid-container {
        grid-template-columns: 1fr;
    }
}
</style>

Título: "Programación en JavaScript para Desarrollo Web"

Recopilación digital introductória al lenguaje de Javascript, aprenderás a utilizarlo para crear interactividad y dinamismo en tus páginas web. A medida que avances, descubrirás cómo manipular el DOM (Document Object Model), responder a eventos y trabajar con datos en JavaScript.

En este capítulo, aprenderás cómo manipular el DOM (Document Object Model) en JavaScript. Descubrirás cómo acceder y modificar elementos HTML utilizando JavaScript.
